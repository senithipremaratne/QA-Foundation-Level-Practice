# Manual Test Cases - QA Foundation Level

## Test Case 1: Login Functionality
- **Objective:** Verify that users can log in with valid credentials
- **Preconditions:** User account exists
- **Steps:**
  1. Open the website
  2. Navigate to Login page
  3. Enter valid username and password
  4. Click "Login"
- **Expected Result:** User is successfully logged in and redirected to dashboard

## Test Case 2: Login with Invalid Credentials
- **Objective:** Verify system handles invalid login correctly
- **Steps:**
  1. Enter invalid username/password
  2. Click "Login"
- **Expected Result:** Error message displayed, user not logged in

## Test Case 3: Navigation Check
- **Objective:** Ensure all main menu links work
- **Steps:**
  1. Click each menu link (Home, About, Contact, etc.)
- **Expected Result:** Correct page loads for each link
